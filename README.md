### About Project
This is a scientific calculator application developed with C# Windows Forms.
Unlike simple calculators, this project can perform trigonometric functions and convert number systems (Decimal to Binary, Hexadecimal, etc.).

### Features
* **Basic Math:** Addition, Subtraction, Multiplication, Division.
* **Scientific Functions:** Sin, Cos, Tan, Log, Ln, Sqrt, Power (x², x³).
* **Number Systems:** It converts numbers between Decimal, Binary, Hexadecimal, and Octal.
* **User Interface:** A simple and easy-to-use design similar to Windows calculator.

### Technologies Used
* C#
* .NET Framework
* Windows Forms Application
